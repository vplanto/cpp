# Основи програмування
## C++ Engineering: Архітектура пам'яті та системне мислення

Цей курс є частиною дисципліни "Основи програмування", яка читається студентам 1-го курсу.

**Фокус курсу:** Ми не просто вчимо синтаксис C++, ми вчимося розуміти, як працює комп'ютер. Курс фокусується на управлінні пам'яттю (Stack vs Heap), роботі з вказівниками, розумінні низькорівневих процесів та написанні високоефективного коду. Це фундамент для майбутніх інженерів та математиків.

Структура курсу:

  1й семестр
  * 1 лекція на тиждень
  * 1 практикум 1 раз тиждень
