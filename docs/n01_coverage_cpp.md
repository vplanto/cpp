# Аналіз покриття програми ЄФВВ та фахових іспитів (113 та 12х) — 1 семестр (C++)

> **⚠️ Статус документа:** Проміжний звіт (1-й семестр)
> Цей аналіз базується на матеріалах **1-го семестру** курсу "Основи програмування (C++)".
>
> Курс є фундаментальним. Він закладає базу для розуміння того, як програмне забезпечення взаємодіє з апаратним, що є критичним для обох напрямків підготовки.

## Вступ: Низький рівень — висока відповідальність

C++ — це мова, яка не пробачає помилок, але дає максимальний контроль над ресурсами комп'ютера. Залежно від вашої спеціальності, мета вивчення цього інструменту відрізняється:

  * **Для 113 (Computational Mathematics):** C++ — це стандарт для **High-Performance Computing (HPC)**. Коли Python занадто повільний для розрахунку моделі, математики переписують код на C++. Ви вчитеся керувати пам'яттю, щоб обробляти величезні матриці та графи.
  * **Для 12x (Software Engineering):** Це "рентген" для розуміння комп'ютера. Вивчаючи C++, ви розумієте, як працюють операційна система, драйвери, ігрові рушії та віртуальні машини інших мов (включно з Java та Python).

## 2\. Специфіка для спеціальності 113 "Прикладна математика"

Курс закриває значну частину блоку **"Чисельні методи"** та **"Програмування"** в частині технічної реалізації.

### Критичні навички для 113:

1.  **Пряме управління пам'яттю:**
      * *Матеріали:* [Лекція 8](08_new.md) (`new`/`delete`), [Практикум 7](p07_new_struct.md).
      * *Чому це важливо:* Математичні моделі часто вимагають створення масивів розмірністю $10^6$ і більше. Розуміння того, як виділити під це пам'ять і не допустити витоків (memory leaks), є базовою вимогою.
2.  **Точність обчислень:**
      * *Матеріали:* [Лекція 4](04_more_on_types.md) (Casting, `double` vs `float`).
      * *Чому це важливо:* Розуміння втрати точності при діленні цілих чисел або переповнення типів є критичним для коректності чисельних методів.
3.  **Алгоритмічна база:**
      * *Матеріали:* [Практикум 0, 2](p00_cycles.md).
      * *Чому це важливо:* Реалізація циклічних алгоритмів (суми рядів, ітераційні процеси) — це щоденна робота прикладного математика.

## 5\. Рекомендації для підготовки (Roadmap)

**Що робити на канікулах / перед 2-м семестром:**

  * **Студентам 113:** Переконайтеся, що ви "на ти" з **вказівниками та масивами** ([Практикум 3](p03_ptr.md), [Практикум 7](p07_new_struct.md)). У наступному семестрі це стане основою для реалізації складних математичних структур (матриць, графів).
  * **Студентам 12x:** Розберіться з концепцією **посилань (`&`) vs вказівників (`*`)** ([Лекція 7](07_references.md)). Це найчастіше питання на співбесідах по C++. Спробуйте самостійно почитати про класи в C++, щоб підготуватися до ООП.
