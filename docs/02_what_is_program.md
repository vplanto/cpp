3.  **–ó–Ω–∞—á–µ–Ω–Ω—è (Value):** –ö–æ–Ω–∫—Ä–µ—Ç–Ω—ñ –¥–∞–Ω—ñ, —â–æ –∑–±–µ—Ä—ñ–≥–∞—é—Ç—å—Å—è —É –∑–º—ñ–Ω–Ω—ñ–π –≤ –¥–∞–Ω–∏–π –º–æ–º–µ–Ω—Ç.

### –û–≥–æ–ª–æ—à–µ–Ω–Ω—è —Ç–∞ —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–∞—Ü—ñ—è –∑–º—ñ–Ω–Ω–∏—Ö

### –ü—Ä–∏—Å–≤–æ—î–Ω–Ω—è —Ç–∞ –∑–º—ñ–Ω–∞ –∑–Ω–∞—á–µ–Ω–Ω—è

–ü—ñ—Å–ª—è –æ–≥–æ–ª–æ—à–µ–Ω–Ω—è –∑–º—ñ–Ω–Ω—ñ–π –º–æ–∂–Ω–∞ –Ω–∞–¥–∞–≤–∞—Ç–∏ –Ω–æ–≤—ñ –∑–Ω–∞—á–µ–Ω–Ω—è –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é **–æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ –ø—Ä–∏—Å–≤–æ—î–Ω–Ω—è (`=`)**.

**–ö–ª—é—á–æ–≤—ñ –ø—Ä–∞–≤–∏–ª–∞ —Ä–æ–±–æ—Ç–∏ –∑—ñ –∑–º—ñ–Ω–Ω–∏–º–∏:**

  * –ó–º—ñ–Ω–Ω–∞ –º–∞—î –±—É—Ç–∏ –æ–≥–æ–ª–æ—à–µ–Ω–∞ **–ø–µ—Ä–µ–¥ —ó—ó –ø–µ—Ä—à–∏–º –≤–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è–º**.
  * –ö–æ–∂–Ω–∞ –∑–º—ñ–Ω–Ω–∞ –º–æ–∂–µ –±—É—Ç–∏ –æ–≥–æ–ª–æ—à–µ–Ω–∞ –ª–∏—à–µ **–æ–¥–∏–Ω —Ä–∞–∑** –≤ –æ–¥–Ω—ñ–π –æ–±–ª–∞—Å—Ç—ñ –≤–∏–¥–∏–º–æ—Å—Ç—ñ.
  * –¢–∏–ø –∑–º—ñ–Ω–Ω–æ—ó **–Ω–µ –º–æ–∂–Ω–∞ –∑–º—ñ–Ω–∏—Ç–∏** –ø—ñ—Å–ª—è –æ–≥–æ–ª–æ—à–µ–Ω–Ω—è.
  * –ó–º—ñ–Ω–Ω–∞ –º–æ–∂–µ –∑–±–µ—Ä—ñ–≥–∞—Ç–∏ –∑–Ω–∞—á–µ–Ω–Ω—è **–ª–∏—à–µ —Å–≤–æ–≥–æ —Ç–∏–ø—É**.

  * –î—É–∂–µ –≤–∞–∂–ª–∏–≤–æ: –ö–æ–∂–µ–Ω –æ–ø–µ—Ä–∞—Ç–æ—Ä –æ–≥–æ–ª–æ—à–µ–Ω–Ω—è —á–∏ –ø—Ä–∏—Å–≤–æ—î–Ω–Ω—è –≤ C++ –º–∞—î –∑–∞–≤–µ—Ä—à—É–≤–∞—Ç–∏—Å—è **–∫—Ä–∞–ø–∫–æ—é –∑ –∫–æ–º–æ—é (`;`)**.

### –ü—Ä–∏—Å–≤–æ—î–Ω–Ω—è (`=`) ‚Äî —Ü–µ –Ω–µ —Ä—ñ–≤–Ω—ñ—Å—Ç—å (`==`)

–í–∫—Ä–∞–π –≤–∞–∂–ª–∏–≤–æ —Ä–æ–∑—Ä—ñ–∑–Ω—è—Ç–∏ –æ–ø–µ—Ä–∞—Ç–æ—Ä –ø—Ä–∏—Å–≤–æ—î–Ω–Ω—è –≤ –ø—Ä–æ–≥—Ä–∞–º—É–≤–∞–Ω–Ω—ñ —Ç–∞ –∑–Ω–∞–∫ —Ä—ñ–≤–Ω–æ—Å—Ç—ñ –≤ –º–∞—Ç–µ–º–∞—Ç–∏—Ü—ñ.

  * –í –º–∞—Ç–µ–º–∞—Ç–∏—Ü—ñ `x = x + 1` ‚Äî —Ü–µ —Ö–∏–±–Ω–µ —Ç–≤–µ—Ä–¥–∂–µ–Ω–Ω—è.
  * –í –ø—Ä–æ–≥—Ä–∞–º—É–≤–∞–Ω–Ω—ñ `x = x + 1;` ‚Äî —Ü–µ –∫–æ–º–∞–Ω–¥–∞: "–æ–±—á–∏—Å–ª–∏—Ç–∏ –∑–Ω–∞—á–µ–Ω–Ω—è `x + 1` —ñ –∑–∞–ø–∏—Å–∞—Ç–∏ –Ω–æ–≤–∏–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç –Ω–∞–∑–∞–¥ —É –∑–º—ñ–Ω–Ω—É `x`".

–ó–∞–≤–∂–¥–∏ —Å–ø–æ—á–∞—Ç–∫—É –æ–±—á–∏—Å–ª—é—î—Ç—å—Å—è –≤–∏—Ä–∞–∑ **—Å–ø—Ä–∞–≤–∞** –≤—ñ–¥ –∑–Ω–∞–∫–∞ `=`, —ñ –ª–∏—à–µ –ø–æ—Ç—ñ–º —Ä–µ–∑—É–ª—å—Ç–∞—Ç –∑–∞–ø–∏—Å—É—î—Ç—å—Å—è —É –∑–º—ñ–Ω–Ω—É **–∑–ª—ñ–≤–∞**. –¢–æ–º—É –∑–∞–ø–∏—Å `x + y = 1;` –≤ C++ —î –Ω–µ–∫–æ—Ä–µ–∫—Ç–Ω–∏–º, –±–æ –∑–ª—ñ–≤–∞ –º–∞—î –±—É—Ç–∏ –ª–∏—à–µ —ñ–º'—è –∑–º—ñ–Ω–Ω–æ—ó.

## –ö–µ—Ä—É–≤–∞–Ω–Ω—è –ø–æ—Ç–æ–∫–æ–º –≤–∏–∫–æ–Ω–∞–Ω–Ω—è

–ü—Ä–æ–≥—Ä–∞–º–∏ –Ω–µ –∑–∞–≤–∂–¥–∏ –≤–∏–∫–æ–Ω—É—é—Ç—å—Å—è –ª—ñ–Ω—ñ–π–Ω–æ. –ß–∞—Å—Ç–æ –ø–æ—Ç—Ä—ñ–±–Ω–æ –ø—Ä–∏–π–º–∞—Ç–∏ —Ä—ñ—à–µ–Ω–Ω—è –∞–±–æ –ø–æ–≤—Ç–æ—Ä—é–≤–∞—Ç–∏ –¥—ñ—ó. –î–ª—è —Ü—å–æ–≥–æ —ñ—Å–Ω—É—é—Ç—å **–∫–µ—Ä—É—é—á—ñ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó**.

### –£–º–æ–≤–∏ (if-else)

–£–º–æ–≤–Ω—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏ –¥–æ–∑–≤–æ–ª—è—é—Ç—å –ø—Ä–æ–≥—Ä–∞–º—ñ –≤–∏–∫–æ–Ω—É–≤–∞—Ç–∏ —Ä—ñ–∑–Ω—ñ –±–ª–æ–∫–∏ –∫–æ–¥—É –∑–∞–ª–µ–∂–Ω–æ –≤—ñ–¥ —Ç–æ–≥–æ, —á–∏ —î –ø–µ–≤–Ω–∞ —É–º–æ–≤–∞ —ñ—Å—Ç–∏–Ω–Ω–æ—é (`true`) —á–∏ —Ö–∏–±–Ω–æ—é (`false`).

```cpp
// –ü—Ä–æ—Å—Ç–∞ —É–º–æ–≤–∞ if
if (number > 0) {
    // —Ü–µ–π –∫–æ–¥ –≤–∏–∫–æ–Ω–∞—î—Ç—å—Å—è, —Ç—ñ–ª—å–∫–∏ —è–∫—â–æ number > 0
    std::cout << "–ß–∏—Å–ª–æ –¥–æ–¥–∞—Ç–Ω–µ";
}

// –£–º–æ–≤–∞ –∑ –∞–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤–æ—é if-else
if (number % 2 == 0) {
    // —Ü–µ–π –±–ª–æ–∫, —è–∫—â–æ —á–∏—Å–ª–æ –ø–∞—Ä–Ω–µ
    std::cout << "–ß–∏—Å–ª–æ –ø–∞—Ä–Ω–µ";
} else {
    // —Ü–µ–π –±–ª–æ–∫, —è–∫—â–æ —á–∏—Å–ª–æ –Ω–µ–ø–∞—Ä–Ω–µ
    std::cout << "–ß–∏—Å–ª–æ –Ω–µ–ø–∞—Ä–Ω–µ";
}
```

–î–ª—è –ø–æ–±—É–¥–æ–≤–∏ —É–º–æ–≤ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å—Å—è **–æ–ø–µ—Ä–∞—Ç–æ—Ä–∏ –ø–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è** —Ç–∞ **–ª–æ–≥—ñ—á–Ω—ñ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏**.

| –û–ø–µ—Ä–∞—Ç–æ—Ä | –û–ø–∏—Å | –ü—Ä–∏–∫–ª–∞–¥ (`a=10, b=5`) |
| :--- | :--- | :--- |
| `>` | –ë—ñ–ª—å—à–µ –Ω—ñ–∂ | `a > b` (true) |
| `<` | –ú–µ–Ω—à–µ –Ω—ñ–∂ | `a < b` (false) |
| `>=` | –ë—ñ–ª—å—à–µ –∞–±–æ –¥–æ—Ä—ñ–≤–Ω—é—î | `a >= 10` (true) |
| `<=` | –ú–µ–Ω—à–µ –∞–±–æ –¥–æ—Ä—ñ–≤–Ω—é—î | `b <= 4` (false) |
| `==` | –î–æ—Ä—ñ–≤–Ω—é—î (–ø–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è) | `a == b` (false) |
| `!=` | –ù–µ –¥–æ—Ä—ñ–≤–Ω—é—î | `a != b` (true) |

| –û–ø–µ—Ä–∞—Ç–æ—Ä | –û–ø–∏—Å | –ü—Ä–∏–∫–ª–∞–¥ (`a=10, b=5`) |
| :--- | :--- | :--- |
| `&&` | –õ–æ–≥—ñ—á–Ω–µ "–Ü" | `(a > 0) && (b == 5)` (true) |
| `\|\|` | –õ–æ–≥—ñ—á–Ω–µ "–ê–ë–û" | `(a > 10) \|\| (b == 5)` (true) |
