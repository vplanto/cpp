#### 2. UTF-8 —Ç–∞ –∫–∏—Ä–∏–ª–∏—Ü—è

–í–∞—à —Ä–µ–¥–∞–∫—Ç–æ—Ä –∫–æ–¥—É (—ñ –∫–æ–º–ø—ñ–ª—è—Ç–æ—Ä) –Ω–∞–π—ñ–º–æ–≤—ñ—Ä–Ω—ñ—à–µ –∑–±–µ—Ä—ñ–≥–∞—î –≤–∞—à `.cpp` —Ñ–∞–π–ª –≤ –∫–æ–¥—É–≤–∞–Ω–Ω—ñ **UTF-8**.

#### 3. –©–æ –±–∞—á–∏—Ç—å `std::string`?
