### ‚ùì –ü–∏—Ç–∞–Ω–Ω—è –¥–æ –≥—Ä—É–ø–∏

–î–∞–≤–∞–π—Ç–µ –ø–æ–º—ñ—Ä–∫—É—î–º–æ: –Ω–∞–≤—ñ—â–æ –º–æ–≤–∏ –ø—Ä–æ–≥—Ä–∞–º—É–≤–∞–Ω–Ω—è —Ä–æ–±–ª—è—Ç—å —á—É—Ç–ª–∏–≤–∏–º–∏ –¥–æ —Ä–µ–≥—ñ—Å—Ç—Ä—É? –¶–µ –∂ —Å—Ç–≤–æ—Ä—é—î –ø–ª—É—Ç–∞–Ω–∏–Ω—É. –ß–∏ —î –≤ —Ü—å–æ–º—É —è–∫–∞—Å—å —ñ–Ω–∂–µ–Ω–µ—Ä–Ω–∞ –ø–µ—Ä–µ–≤–∞–≥–∞?

## 3\. –†–æ–±–æ—Ç–∞ –∑ –¥–∞–Ω–∏–º–∏: –ó–º—ñ–Ω–Ω—ñ —Ç–∞ —Ñ—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω—ñ —Ç–∏–ø–∏

C++ ‚Äî —Ü–µ **—Å—Ç–∞—Ç–∏—á–Ω–æ —Ç–∏–ø—ñ–∑–æ–≤–∞–Ω–∞** –º–æ–≤–∞.  –¶–µ –æ–∑–Ω–∞—á–∞—î, —â–æ –¥–ª—è –∫–æ–∂–Ω–æ—ó –∑–º—ñ–Ω–Ω–æ—ó (—ñ–º–µ–Ω–æ–≤–∞–Ω–æ—ó –∫–æ–º—ñ—Ä–∫–∏ –ø–∞–º'—è—Ç—ñ) –º–∏ –ø–æ–≤–∏–Ω–Ω—ñ –∑–∞–∑–¥–∞–ª–µ–≥—ñ–¥—å –≤–∫–∞–∑–∞—Ç–∏, —è–∫–∏–π —Ç–∏–ø –¥–∞–Ω–∏—Ö –≤–æ–Ω–∞ –±—É–¥–µ –∑–±–µ—Ä—ñ–≥–∞—Ç–∏. 

### –§—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω—ñ —Ç–∏–ø–∏ –¥–∞–Ω–∏—Ö

–¶–µ –±–∞–∑–æ–≤—ñ "—Ü–µ–≥–ª–∏–Ω–∫–∏" –¥–ª—è –¥–∞–Ω–∏—Ö.

| –¢–∏–ø | –û–ø–∏—Å | –†–æ–∑–º—ñ—Ä —É –±–∞–π—Ç–∞—Ö (—Ç–∏–ø–æ–≤–∏–π) | –ü—Ä–∏–∫–ª–∞–¥ –∫–æ–¥—É (–∑ –∞–∫—Ç—É–∞–ª—å–Ω–∏–º –∫–æ–Ω—Ç–µ–∫—Å—Ç–æ–º) |
| :--- | :--- | :--- | :--- |
| `int` | **–¶—ñ–ª–µ —á–∏—Å–ª–æ**  | 4 | `int drones_count = 15;` |
| `float` | **–î—ñ–π—Å–Ω–µ —á–∏—Å–ª–æ** (–æ–¥–∏–Ω–∞—Ä–Ω–æ—ó —Ç–æ—á–Ω–æ—Å—Ç—ñ)  | 4 | `float core_temp = 80.5f;` |
| `double` | **–î—ñ–π—Å–Ω–µ —á–∏—Å–ª–æ** (–ø–æ–¥–≤—ñ–π–Ω–æ—ó —Ç–æ—á–Ω–æ—Å—Ç—ñ)  | 8 | `double donation_amount = 250.75;` |

### –ß–∏—Å–ª–∞ –∑ –ø–ª–∞–≤–∞—é—á–æ—é –∫–æ–º–æ—é: `float` vs. `double`

–û—Å–Ω–æ–≤–Ω–∞ –≤—ñ–¥–º—ñ–Ω–Ω—ñ—Å—Ç—å ‚Äî —É **—Ç–æ—á–Ω–æ—Å—Ç—ñ** —Ç–∞ **–¥—ñ–∞–ø–∞–∑–æ–Ω—ñ** –∑–Ω–∞—á–µ–Ω—å. 

  * **`float` (–û–¥–∏–Ω–∞—Ä–Ω–∞ —Ç–æ—á–Ω—ñ—Å—Ç—å):** –†–æ–∑–º—ñ—Ä 4 –±–∞–π—Ç–∏. –¢–æ—á–Ω—ñ—Å—Ç—å \~7 –¥–µ—Å—è—Ç–∫–æ–≤–∏—Ö –∑–Ω–∞–∫—ñ–≤. 
  * **`double` (–ü–æ–¥–≤—ñ–π–Ω–∞ —Ç–æ—á–Ω—ñ—Å—Ç—å):** –†–æ–∑–º—ñ—Ä 8 –±–∞–π—Ç. –¢–æ—á–Ω—ñ—Å—Ç—å \~15-17 –¥–µ—Å—è—Ç–∫–æ–≤–∏—Ö –∑–Ω–∞–∫—ñ–≤. 

> **–ê–Ω–∞–ª–æ–≥—ñ—è:** üí° –£—è–≤—ñ—Ç—å, —â–æ `float` ‚Äî —Ü–µ –∑–≤–∏—á–∞–π–Ω–∞ –ª—ñ–Ω—ñ–π–∫–∞, —è–∫–∞ –¥–æ–∑–≤–æ–ª—è—î –≤–∏–º—ñ—Ä—è—Ç–∏ –¥–æ–≤–∂–∏–Ω—É –∑ —Ç–æ—á–Ω—ñ—Å—Ç—é –¥–æ –º—ñ–ª—ñ–º–µ—Ç—Ä–∞. `double` ‚Äî —Ü–µ –ª–∞–∑–µ—Ä–Ω–∏–π –≤–∏–º—ñ—Ä—é–≤–∞—á, —â–æ –¥–∞—î –∑–Ω–∞—á–Ω–æ —Ç–æ—á–Ω—ñ—à–∏–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç.

**–ó–∞–≤–∂–¥–∏ –Ω–∞–¥–∞–≤–∞–π—Ç–µ –ø–µ—Ä–µ–≤–∞–≥—É `double`**, —è–∫—â–æ —É –≤–∞—Å –Ω–µ–º–∞—î –≤–∞–≥–æ–º–∏—Ö –ø—Ä–∏—á–∏–Ω –µ–∫–æ–Ω–æ–º–∏—Ç–∏ –ø–∞–º'—è—Ç—å (–Ω–∞–ø—Ä–∏–∫–ª–∞–¥, —É –∫–æ–º–ø'—é—Ç–µ—Ä–Ω—ñ–π –≥—Ä–∞—Ñ—ñ—Ü—ñ).

### ‚ùì –ü–∏—Ç–∞–Ω–Ω—è –¥–æ –≥—Ä—É–ø–∏

–£—è–≤—ñ—Ç—å, —â–æ –º–∏ –ø–∏—à–µ–º–æ –ø—Ä–æ–≥—Ä–∞–º—É –¥–ª—è –±–∞–ª—ñ—Å—Ç–∏—á–Ω–æ–≥–æ –∫–∞–ª—å–∫—É–ª—è—Ç–æ—Ä–∞. –Ø–∫–∏–π —Ç–∏–ø –≤–∏ –æ–±–µ—Ä–µ—Ç–µ –¥–ª—è —Ä–æ–∑—Ä–∞—Ö—É–Ω–∫—É —Ç—Ä–∞—î–∫—Ç–æ—Ä—ñ—ó, `float` —á–∏ `double`, —ñ —á–æ–º—É?

## 4\. –í–≤—ñ–¥ —Ç–∞ –≤–∏–≤—ñ–¥ –¥–∞–Ω–∏—Ö (I/O)

### –í–∏–≤—ñ–¥ –¥–∞–Ω–∏—Ö –∑ `cout`

–ú–∏ –≤–∂–µ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞–ª–∏ `cout` –¥–ª—è –≤–∏–≤–æ–¥—É.  –û–ø–µ—Ä–∞—Ç–æ—Ä `<<` –º–æ–∂–Ω–∞ –ø–æ—î–¥–Ω—É–≤–∞—Ç–∏ –≤ –ª–∞–Ω—Ü—é–∂–æ–∫. 

```cpp
string drone_model = "Mavic 3T";

cout << "Model: " << drone_model << ", Quantity: " << quantity << endl;
```

### `endl` —á–∏ `'\n'`: –£ —á–æ–º—É –Ω–∞—Å–ø—Ä–∞–≤–¥—ñ —Ä—ñ–∑–Ω–∏—Ü—è?

–û–±–∏–¥–≤–∞ —Å–∏–º–≤–æ–ª–∏ –ø–æ—á–∏–Ω–∞—é—Ç—å –Ω–æ–≤–∏–π —Ä—è–¥–æ–∫. 

  * **`'\n'`** ‚Äî —Ü–µ –ø—Ä–æ—Å—Ç–æ —Å–∏–º–≤–æ–ª –Ω–æ–≤–æ–≥–æ —Ä—è–¥–∫–∞. 
  * **`std::endl`** ‚Äî —Ü–µ –º–∞–Ω—ñ–ø—É–ª—è—Ç–æ—Ä, —è–∫–∏–π —Ä–æ–±–∏—Ç—å **–¥–≤—ñ —Ä–µ—á—ñ**:
    1.  –í—Å—Ç–∞–≤–ª—è—î —Å–∏–º–≤–æ–ª –Ω–æ–≤–æ–≥–æ —Ä—è–¥–∫–∞ `'\n'`. 
    2.  **–û—á–∏—â—É—î –±—É—Ñ–µ—Ä (flushing)**, —Ç–æ–±—Ç–æ –≥–∞—Ä–∞–Ω—Ç–æ–≤–∞–Ω–æ –∑–º—É—à—É—î –ø—Ä–æ–≥—Ä–∞–º—É –Ω–µ–≥–∞–π–Ω–æ –≤–∏–≤–µ—Å—Ç–∏ –≤—Å—ñ –¥–∞–Ω—ñ –Ω–∞ –µ–∫—Ä–∞–Ω. 

–ß–∞—Å—Ç–µ –æ—á–∏—â–µ–Ω–Ω—è –±—É—Ñ–µ—Ä–∞ (flushing) –º–æ–∂–µ —Å–ø–æ–≤—ñ–ª—å–Ω—é–≤–∞—Ç–∏ –ø—Ä–æ–≥—Ä–∞–º—É, –æ—Å–æ–±–ª–∏–≤–æ –ø—Ä–∏ –∑–∞–ø–∏—Å—ñ —É —Ñ–∞–π–ª.

**–†–µ–∫–æ–º–µ–Ω–¥–∞—Ü—ñ—è:** –í–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–π—Ç–µ `'\n'` –¥–ª—è –±—ñ–ª—å—à–æ—Å—Ç—ñ –≤–∏–ø–∞–¥–∫—ñ–≤, –æ—Å–æ–±–ª–∏–≤–æ –≤—Å–µ—Ä–µ–¥–∏–Ω—ñ –¥–æ–≤–≥–∏—Ö —Ä—è–¥–∫—ñ–≤. –í–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–π—Ç–µ `std::endl` —Ç—ñ–ª—å–∫–∏ —Ç–æ–¥—ñ, –∫–æ–ª–∏ –≤–∞–º –ø–æ—Ç—Ä—ñ–±–Ω–æ *–≥–∞—Ä–∞–Ω—Ç–æ–≤–∞–Ω–æ* –ø–æ–±–∞—á–∏—Ç–∏ –≤–∏–≤—ñ–¥ –Ω–µ–≥–∞–π–Ω–æ (–Ω–∞–ø—Ä–∏–∫–ª–∞–¥, –ø–µ—Ä–µ–¥ `cin` –∞–±–æ –ø—ñ–¥ —á–∞—Å –≤—ñ–¥–ª–∞–¥–∫–∏).

### –í–≤—ñ–¥ –¥–∞–Ω–∏—Ö –∑ `cin`

–î–ª—è –æ—Ç—Ä–∏–º–∞–Ω–Ω—è –¥–∞–Ω–∏—Ö –≤—ñ–¥ –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è –ø–æ—Ç—ñ–∫ –≤–≤–æ–¥—É `cin` —Ç–∞ –æ–ø–µ—Ä–∞—Ç–æ—Ä `>>`. 

**–ü—Ä–∏–∫–ª–∞–¥: –ú—ñ–Ω—ñ-—É—Ç–∏–ª—ñ—Ç–∞ "–ó–∞–ª–∏—à–æ–∫ –∑–±–æ—Ä—É"**

```cpp
#include <iostream>

using namespace std;

int main() {
    // 1. –û–≥–æ–ª–æ—à—É—î–º–æ –∑–º—ñ–Ω–Ω—ñ
    int drones_needed;
    int drones_purchased;
    int drones_remaining;

    // 2. –ü—Ä–æ—Å–∏–º–æ –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞ –≤–≤–µ—Å—Ç–∏ –¥–∞–Ω—ñ
    cout << "Enter total drones needed: "; 
    
    // 3. –ß–∏—Ç–∞—î–º–æ –≤–≤–µ–¥–µ–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è 
    cin >> drones_needed; 

    // 4. –†–æ–±–∏–º–æ —Ç–µ –∂ —Å–∞–º–µ –¥–ª—è –¥—Ä—É–≥–æ–≥–æ —á–∏—Å–ª–∞

    cin >> drones_purchased;

    // 5. –û–±—á–∏—Å–ª—é—î–º–æ –∑–∞–ª–∏—à–æ–∫
    drones_remaining = drones_needed - drones_purchased;

    // 6. –í–∏–≤–æ–¥–∏–º–æ —Ä–µ–∑—É–ª—å—Ç–∞—Ç
    cout << "Remaining to collect: " << drones_remaining << " drones." << endl;

    return 0;
}
```

–ú–æ–∂–Ω–∞ –∑—á–∏—Ç—É–≤–∞—Ç–∏ –∫—ñ–ª—å–∫–∞ –∑–Ω–∞—á–µ–Ω—å –æ–¥–Ω–æ—á–∞—Å–Ω–æ. –ö–æ—Ä–∏—Å—Ç—É–≤–∞—á –º–∞—î –≤–≤–æ–¥–∏—Ç–∏ —ó—Ö —á–µ—Ä–µ–∑ –ø—Ä–æ–±—ñ–ª.

## 5\. –°—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–∞ –±—ñ–±–ª—ñ–æ—Ç–µ–∫–∞ —à–∞–±–ª–æ–Ω—ñ–≤ (STL)

**STL** (Standard Template Library) ‚Äî —Ü–µ –ø–æ—Ç—É–∂–Ω–∞ —á–∞—Å—Ç–∏–Ω–∞ C++, —è–∫–∞ –Ω–∞–¥–∞—î –≥–æ—Ç–æ–≤—ñ —Ä—ñ—à–µ–Ω–Ω—è. 

  * –í–æ–Ω–∞ –º—ñ—Å—Ç–∏—Ç—å –≥–æ—Ç–æ–≤—ñ –∫–ª–∞—Å–∏, —è–∫ `vector` (–¥–∏–Ω–∞–º—ñ—á–Ω–∏–π –º–∞—Å–∏–≤), `map` (—Å–ª–æ–≤–Ω–∏–∫) —Ç–∞ –±–∞–≥–∞—Ç–æ —ñ–Ω—à–∏—Ö. 
  * –£—Å—ñ –µ–ª–µ–º–µ–Ω—Ç–∏ STL –∑–Ω–∞—Ö–æ–¥—è—Ç—å—Å—è —É –ø—Ä–æ—Å—Ç–æ—Ä—ñ —ñ–º–µ–Ω `std`.  –°–∞–º–µ —Ç–æ–º—É –º–∏ –ø–∏—à–µ–º–æ `std::cout` –∞–±–æ `std::string`.
  * –î–∏—Ä–µ–∫—Ç–∏–≤–∞ `using namespace std;` –¥–æ–∑–≤–æ–ª—è—î –Ω–µ –ø–∏—Å–∞—Ç–∏ –ø—Ä–µ—Ñ—ñ–∫—Å `std::` —â–æ—Ä–∞–∑—É. 

### ‚ùì –ü–∏—Ç–∞–Ω–Ω—è –¥–æ –≥—Ä—É–ø–∏

–ú–∏ –∫–∞–∂–µ–º–æ, —â–æ STL "–¥–æ–±—Ä–µ –æ–ø—Ç–∏–º—ñ–∑–æ–≤–∞–Ω–∞".  –©–æ —Ü–µ –æ–∑–Ω–∞—á–∞—î –¥–ª—è –Ω–∞—Å, —è–∫ –¥–ª—è —Ä–æ–∑—Ä–æ–±–Ω–∏–∫—ñ–≤, –Ω–∞ –ø—Ä–∞–∫—Ç–∏—Ü—ñ? –ß–æ–º—É –∫—Ä–∞—â–µ –≤–∑—è—Ç–∏ –≥–æ—Ç–æ–≤–∏–π `std::vector`, –Ω—ñ–∂ –ø–∏—Å–∞—Ç–∏ —Å–≤—ñ–π "–≤–µ–ª–æ—Å–∏–ø–µ–¥"?

## 6\. –ó–∞–≤–¥–∞–Ω–Ω—è –¥–ª—è —Å–∞–º–æ—Å—Ç—ñ–π–Ω–æ—ó —Ä–æ–±–æ—Ç–∏

1.  **–ö–æ–Ω–≤–µ—Ä—Ç–µ—Ä –≤–∞–ª—é—Ç**: –ù–∞–ø–∏—à—ñ—Ç—å –ø—Ä–æ–≥—Ä–∞–º—É, —è–∫–∞ –∑–∞–ø–∏—Ç—É—î —É –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞ —Å—É–º—É –≤ –≥—Ä–∏–≤–Ω—è—Ö (UAH) —Ç–∞ –ø–æ—Ç–æ—á–Ω–∏–π –∫—É—Ä—Å –¥–æ–ª–∞—Ä–∞ (–Ω–∞–ø—Ä–∏–∫–ª–∞–¥, 39.5). –ü—Ä–æ–≥—Ä–∞–º–∞ –º–∞—î –æ–±—á–∏—Å–ª–∏—Ç–∏ —Ç–∞ –≤–∏–≤–µ—Å—Ç–∏, —Å–∫—ñ–ª—å–∫–∏ –¥–æ–ª–∞—Ä—ñ–≤ (USD) –º–æ–∂–Ω–∞ –∫—É–ø–∏—Ç–∏ –∑–∞ —Ü—é —Å—É–º—É. –í–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–π—Ç–µ —Ç–∏–ø `double`.
2.  **–†–æ–∑—Ä–∞—Ö—É–Ω–æ–∫ –∑–±–æ—Ä—É**: –ù–∞–ø–∏—à—ñ—Ç—å –ø—Ä–æ–≥—Ä–∞–º—É, —è–∫–∞ –∑–∞–ø–∏—Ç—É—î —Ü—ñ–Ω—É –æ–¥–Ω–æ–≥–æ –¥—Ä–æ–Ω–∞ (`double`) —Ç–∞ —ó—Ö–Ω—é –∫—ñ–ª—å–∫—ñ—Å—Ç—å (`int`). –ü—Ä–æ–≥—Ä–∞–º–∞ –º–∞—î –æ–±—á–∏—Å–ª–∏—Ç–∏ —Ç–∞ –≤–∏–≤–µ—Å—Ç–∏ –∑–∞–≥–∞–ª—å–Ω—É —Å—É–º—É –∑–±–æ—Ä—É.
3.  **–°—Ç–∞—Ç–∏—Å—Ç–∏–∫–∞ –≤–æ–ª–æ–Ω—Ç–µ—Ä—ñ–≤**: –ù–∞–ø–∏—à—ñ—Ç—å –ø—Ä–æ–≥—Ä–∞–º—É, —è–∫–∞ –∑–∞–ø–∏—Ç—É—î, —Å–∫—ñ–ª—å–∫–∏ –≤–æ–ª–æ–Ω—Ç–µ—Ä—ñ–≤ –ø—Ä–∞—Ü—é–≤–∞–ª–æ –≤—Ä–∞–Ω—Ü—ñ (`int morning_shift`) —Ç–∞ —Å–∫—ñ–ª—å–∫–∏ –≤–≤–µ—á–µ—Ä—ñ (`int evening_shift`). –ü—Ä–æ–≥—Ä–∞–º–∞ –º–∞—î –≤–∏–≤–µ—Å—Ç–∏ –∑–∞–≥–∞–ª—å–Ω—É –∫—ñ–ª—å–∫—ñ—Å—Ç—å –≤–æ–ª–æ–Ω—Ç–µ—Ä—ñ–≤ –∑–∞ –¥–µ–Ω—å.

## 7\. –ö–ª—é—á–æ–≤—ñ –º–æ–º–µ–Ω—Ç–∏ —Ç–∞ –ø–æ—Ä–∞–¥–∏

  * **–ó–∞–≤–∂–¥–∏ –∑–∞–≤–µ—Ä—à—É–π—Ç–µ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó –∫—Ä–∞–ø–∫–æ—é –∑ –∫–æ–º–æ—é (`;`)**.
