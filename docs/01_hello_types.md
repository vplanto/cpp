# –í—Å—Ç—É–ø –¥–æ C++: –¢–∏–ø–∏ —Ç–∞ –ø–µ—Ä—à–∞ –ø—Ä–æ–≥—Ä–∞–º–∞

–í—ñ—Ç–∞—é! –¶–µ–π –¥–æ–∫—É–º–µ–Ω—Ç —î –≤—Å—Ç—É–ø–Ω–æ—é –ª–µ–∫—Ü—ñ—î—é –∑ C++. –ú–∏ —Ä–æ–∑–≥–ª—è–Ω–µ–º–æ –±–∞–∑–æ–≤—ñ –∫–æ–Ω—Ü–µ–ø—Ü—ñ—ó, —Å—Ç–≤–æ—Ä–∏–º–æ –Ω–∞—à—É –ø–µ—Ä—à—É –ø—Ä–æ–≥—Ä–∞–º—É —Ç–∞ –Ω–∞–≤—á–∏–º–æ—Å—è –ø—Ä–∞—Ü—é–≤–∞—Ç–∏ –∑ –æ—Å–Ω–æ–≤–Ω–∏–º–∏ —Ç–∏–ø–∞–º–∏ –¥–∞–Ω–∏—Ö.

## 1. –ù–∞—à–∞ –ø–µ—Ä—à–∞ –ø—Ä–æ–≥—Ä–∞–º–∞: "Hello, World!"

–¶–µ —Ç—Ä–∞–¥–∏—Ü—ñ–π–Ω–∏–π –ø–µ—Ä—à–∏–π –∫—Ä–æ–∫ —É –≤–∏–≤—á–µ–Ω–Ω—ñ –±—É–¥—å-—è–∫–æ—ó –º–æ–≤–∏ –ø—Ä–æ–≥—Ä–∞–º—É–≤–∞–Ω–Ω—è. –ú–∏ —Å—Ç–≤–æ—Ä–∏–º–æ –ø—Ä–æ–≥—Ä–∞–º—É, —è–∫–∞ –ø—Ä–æ—Å—Ç–æ –≤–∏–≤–æ–¥–∏—Ç—å —Ç–µ–∫—Å—Ç "hello, world" –Ω–∞ –µ–∫—Ä–∞–Ω.

### –ö—Ä–æ–∫ 1: –ù–∞–ø–∏—Å–∞–Ω–Ω—è –≤–∏—Ö—ñ–¥–Ω–æ–≥–æ –∫–æ–¥—É

–ö–æ–¥ –ø–æ—Ç—Ä—ñ–±–Ω–æ –ø–∏—Å–∞—Ç–∏ —É —Ç–µ–∫—Å—Ç–æ–≤–æ–º—É —Ä–µ–¥–∞–∫—Ç–æ—Ä—ñ –∞–±–æ —Å–µ—Ä–µ–¥–æ–≤–∏—â—ñ —Ä–æ–∑—Ä–æ–±–∫–∏ (IDE). –§–∞–π–ª —Å–ª—ñ–¥ –∑–±–µ—Ä–µ–≥—Ç–∏ –∑ —Ä–æ–∑—à–∏—Ä–µ–Ω–Ω—è–º `.cpp`, –Ω–∞–ø—Ä–∏–∫–ª–∞–¥, `hello.cpp`.

```cpp

 * –ü–µ—Ä—à–∞ –ø—Ä–æ–≥—Ä–∞–º–∞ –Ω–∞ C++, —è–∫–∞ –≤–∏–≤–æ–¥–∏—Ç—å "hello, world" (hello.cpp)

    // cout ‚Äî —Ü–µ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–∏–π –ø–æ—Ç—ñ–∫ –≤–∏–≤–æ–¥—É (–∑–∞–∑–≤–∏—á–∞–π, –≤–∞—à –µ–∫—Ä–∞–Ω)
    // << ‚Äî —Ü–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä "–≤—Å—Ç–∞–≤–∫–∏ –≤ –ø–æ—Ç—ñ–∫", –≤—ñ–Ω –Ω–∞–ø—Ä–∞–≤–ª—è—î –¥–∞–Ω—ñ –≤ cout

````

### –ö—Ä–æ–∫ 2: –ö–æ–º–ø—ñ–ª—è—Ü—ñ—è —Ç–∞ –∑–∞–ø—É—Å–∫

–ù–∞–ø–∏—Å–∞–Ω–∏–π –Ω–∞–º–∏ –∫–æ–¥ ‚Äî —Ü–µ –ø—Ä–æ—Å—Ç–æ —Ç–µ–∫—Å—Ç. –©–æ–± –∫–æ–º–ø'—é—Ç–µ—Ä –º—ñ–≥ –π–æ–≥–æ –≤–∏–∫–æ–Ω–∞—Ç–∏, –π–æ–≥–æ –ø–æ—Ç—Ä—ñ–±–Ω–æ **—Å–∫–æ–º–ø—ñ–ª—é–≤–∞—Ç–∏** ‚Äî –ø–µ—Ä–µ—Ç–≤–æ—Ä–∏—Ç–∏ –Ω–∞ –º–∞—à–∏–Ω–Ω–∏–π –∫–æ–¥. –î–ª—è —Ü—å–æ–≥–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è —Å–ø–µ—Ü—ñ–∞–ª—å–Ω–∞ –ø—Ä–æ–≥—Ä–∞–º–∞ ‚Äî –∫–æ–º–ø—ñ–ª—è—Ç–æ—Ä (–Ω–∞–ø—Ä–∏–∫–ª–∞–¥, `g++`).

1.  **–ö–æ–º–ø—ñ–ª—è—Ü—ñ—è (Build)**: –°—Ç–≤–æ—Ä—é—î –∑ —Ñ–∞–π–ª—É `hello.cpp` –≤–∏–∫–æ–Ω—É–≤–∞–Ω–∏–π —Ñ–∞–π–ª (`hello.exe` –¥–ª—è Windows –∞–±–æ `hello` –¥–ª—è Linux/macOS).


2.  **–ó–∞–ø—É—Å–∫ (Run)**: –í–∏–∫–æ–Ω—É—î–º–æ —Å—Ç–≤–æ—Ä–µ–Ω–∏–π —Ñ–∞–π–ª.

    ```bash

    # –î–ª—è Linux / macOS (./ –æ–∑–Ω–∞—á–∞—î "–≤ –ø–æ—Ç–æ—á–Ω—ñ–π –¥–∏—Ä–µ–∫—Ç–æ—Ä—ñ—ó")

    ```

–†–µ–∑—É–ª—å—Ç–∞—Ç–æ–º –≤–∏–∫–æ–Ω–∞–Ω–Ω—è –±—É–¥–µ —Ç–µ–∫—Å—Ç `hello, world` –Ω–∞ –µ–∫—Ä–∞–Ω—ñ.

### –ü–æ–≤–Ω–∏–π –ø—Ä–æ—Ü–µ—Å: –í—ñ–¥ –∫–æ–¥—É –¥–æ –ø—Ä–æ–≥—Ä–∞–º–∏

–©–æ–± –∫—Ä–∞—â–µ –∑—Ä–æ–∑—É–º—ñ—Ç–∏, —è–∫ –∑–≤–∏—á–∞–π–Ω–∏–π —Ç–µ–∫—Å—Ç –ø–µ—Ä–µ—Ç–≤–æ—Ä—é—î—Ç—å—Å—è –Ω–∞ —Ä–æ–±–æ—á—É –ø—Ä–æ–≥—Ä–∞–º—É, —Ä–æ–∑–≥–ª—è–Ω–µ–º–æ –ø–æ–≤–Ω–∏–π —à–ª—è—Ö —ó—ó —Å—Ç–≤–æ—Ä–µ–Ω–Ω—è:

1.  **–ù–∞–ø–∏—Å–∞–Ω–Ω—è –∫–æ–¥—É (Editor/IDE)**: –í–∏ –ø–∏—à–µ—Ç–µ –∫–æ–¥ —É —Ñ–∞–π–ª–∞—Ö `.cpp` —Ç–∞ `.h`.
2.  **–ü—Ä–µ–ø—Ä–æ—Ü–µ—Å–æ—Ä (Preprocessor)**: –ü–µ—Ä—à –Ω—ñ–∂ –∫–æ–º–ø—ñ–ª—è—Ç–æ—Ä –ø–æ—á–Ω–µ —Ä–æ–±–æ—Ç—É, –ø—Ä–µ–ø—Ä–æ—Ü–µ—Å–æ—Ä –æ–±—Ä–æ–±–ª—è—î –¥–∏—Ä–µ–∫—Ç–∏–≤–∏, —â–æ –ø–æ—á–∏–Ω–∞—é—Ç—å—Å—è –∑ `#` (–Ω–∞–ø—Ä–∏–∫–ª–∞–¥, `#include <iostream>`). –í—ñ–Ω –±—É–∫–≤–∞–ª—å–Ω–æ –≤—Å—Ç–∞–≤–ª—è—î –≤–º—ñ—Å—Ç —ñ–Ω—à–∏—Ö —Ñ–∞–π–ª—ñ–≤ —É –≤–∞—à –∫–æ–¥.
3.  **–ö–æ–º–ø—ñ–ª—è—Ç–æ—Ä (Compiler)**: –ö–æ–º–ø—ñ–ª—è—Ç–æ—Ä –ø–µ—Ä–µ—Ç–≤–æ—Ä—é—î –æ–±—Ä–æ–±–ª–µ–Ω–∏–π –ø—Ä–µ–ø—Ä–æ—Ü–µ—Å–æ—Ä–æ–º –∫–æ–¥ –Ω–∞ –æ–±'—î–∫—Ç–Ω—ñ —Ñ–∞–π–ª–∏ (`.obj` –∞–±–æ `.o`). –¶–µ –≤–∂–µ –ø—Ä–æ–º—ñ–∂–Ω–∏–π –º–∞—à–∏–Ω–Ω–∏–π –∫–æ–¥, –∞–ª–µ –≤—ñ–Ω —â–µ –Ω–µ —î –ø–æ–≤–Ω–æ—Ü—ñ–Ω–Ω–æ—é –ø—Ä–æ–≥—Ä–∞–º–æ—é.
4.  **–õ—ñ–Ω–∫–µ—Ä (Linker)**: –õ—ñ–Ω–∫–µ—Ä (–∫–æ–º–ø–æ–Ω—É–≤–∞–ª—å–Ω–∏–∫) –∑–±–∏—Ä–∞—î –≤–∞—à –æ–±'—î–∫—Ç–Ω–∏–π –∫–æ–¥, –∫–æ–¥ –∑—ñ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–∏—Ö –±—ñ–±–ª—ñ–æ—Ç–µ–∫ (—è–∫ `iostream`) —Ç–∞ —ñ–Ω—à–∏—Ö —Ñ–∞–π–ª—ñ–≤ —Ä–∞–∑–æ–º, —â–æ–± —Å—Ç–≤–æ—Ä–∏—Ç–∏ —î–¥–∏–Ω–∏–π –≤–∏–∫–æ–Ω—É–≤–∞–Ω–∏–π —Ñ–∞–π–ª (`.exe` –∞–±–æ –±–µ–∑ —Ä–æ–∑—à–∏—Ä–µ–Ω–Ω—è).
5.  **–ó–∞–≤–∞–Ω—Ç–∞–∂—É–≤–∞—á (Loader)**: –ö–æ–ª–∏ –≤–∏ –∑–∞–ø—É—Å–∫–∞—î—Ç–µ –ø—Ä–æ–≥—Ä–∞–º—É, –∑–∞–≤–∞–Ω—Ç–∞–∂—É–≤–∞—á –æ–ø–µ—Ä–∞—Ü—ñ–π–Ω–æ—ó —Å–∏—Å—Ç–µ–º–∏ –∫–æ–ø—ñ—é—î —ó—ó –≤ –æ–ø–µ—Ä–∞—Ç–∏–≤–Ω—É –ø–∞–º'—è—Ç—å.
6.  **–í–∏–∫–æ–Ω–∞–Ω–Ω—è (CPU)**: –¶–µ–Ω—Ç—Ä–∞–ª—å–Ω–∏–π –ø—Ä–æ—Ü–µ—Å–æ—Ä (CPU) –ø–æ—á–∏–Ω–∞—î –≤–∏–∫–æ–Ω—É–≤–∞—Ç–∏ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó –∑ –≤–∞—à–æ—ó –ø—Ä–æ–≥—Ä–∞–º–∏.

## 2\. –û—Å–Ω–æ–≤–∏ —Å–∏–Ω—Ç–∞–∫—Å–∏—Å—É C++

  * **–Ü–Ω—Å—Ç—Ä—É–∫—Ü—ñ—è (Statement)**: –¶–µ –æ–∫—Ä–µ–º–∞ –¥—ñ—è, –Ω–∞–ø—Ä–∏–∫–ª–∞–¥, `cout << "hello";` –∞–±–æ `return 0;`. –ö–æ–∂–Ω–∞ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—è –≤ C++ –º–∞—î –∑–∞–∫—ñ–Ω—á—É–≤–∞—Ç–∏—Å—è –∫—Ä–∞–ø–∫–æ—é –∑ –∫–æ–º–æ—é (`;`).
  * **–î–∏—Ä–µ–∫—Ç–∏–≤–∞ –ø—Ä–µ–ø—Ä–æ—Ü–µ—Å–æ—Ä–∞**: –ö–æ–º–∞–Ω–¥–∏ –¥–ª—è –ø—Ä–µ–ø—Ä–æ—Ü–µ—Å–æ—Ä–∞, —â–æ –ø–æ—á–∏–Ω–∞—é—Ç—å—Å—è –∑ `#` (–Ω–∞–ø—Ä–∏–∫–ª–∞–¥, `#include`). –í–æ–Ω–∏ **–Ω–µ** –∑–∞–∫—ñ–Ω—á—É—é—Ç—å—Å—è –∫—Ä–∞–ø–∫–æ—é –∑ –∫–æ–º–æ—é.
  * **–ë–ª–æ–∫ –∫–æ–¥—É**: –ì—Ä—É–ø–∞ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ–π, –≤–∑—è—Ç–∞ —É —Ñ—ñ–≥—É—Ä–Ω—ñ –¥—É–∂–∫–∏ `{ }`. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, —Ç—ñ–ª–æ —Ñ—É–Ω–∫—Ü—ñ—ó `main`. –ü—ñ—Å–ª—è –∑–∞–∫—Ä–∏–≤–∞—é—á–æ—ó –¥—É–∂–∫–∏ `;` –Ω–µ —Å—Ç–∞–≤–∏—Ç—å—Å—è.
  * **–ö–æ–º–µ–Ω—Ç–∞—Ä—ñ**: –¢–µ–∫—Å—Ç, —è–∫–∏–π –∫–æ–º–ø—ñ–ª—è—Ç–æ—Ä —ñ–≥–Ω–æ—Ä—É—î. –ü–æ—Ç—Ä—ñ–±–µ–Ω –¥–ª—è –ø–æ—è—Å–Ω–µ–Ω—å —É –∫–æ–¥—ñ.
      * `// –û–¥–Ω–æ—Ä—è–¥–∫–æ–≤–∏–π –∫–æ–º–µ–Ω—Ç–∞—Ä`
      * `/* –ë–∞–≥–∞—Ç–æ—Ä—è–¥–∫–æ–≤–∏–π –∫–æ–º–µ–Ω—Ç–∞—Ä */`
  * **–ß—É—Ç–ª–∏–≤—ñ—Å—Ç—å –¥–æ —Ä–µ–≥—ñ—Å—Ç—Ä—É**: C++ —Ä–æ–∑—Ä—ñ–∑–Ω—è—î –≤–µ–ª–∏–∫—ñ —Ç–∞ –º–∞–ª—ñ –ª—ñ—Ç–µ—Ä–∏. `main`, `Main` —Ç–∞ `MAIN` ‚Äî —Ü–µ —Ç—Ä–∏ —Ä—ñ–∑–Ω—ñ —ñ–¥–µ–Ω—Ç–∏—Ñ—ñ–∫–∞—Ç–æ—Ä–∏.

## 3\. –†–æ–±–æ—Ç–∞ –∑ –¥–∞–Ω–∏–º–∏: –ó–º—ñ–Ω–Ω—ñ —Ç–∞ —Ñ—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω—ñ —Ç–∏–ø–∏

C++ ‚Äî —Ü–µ **—Å—Ç–∞—Ç–∏—á–Ω–æ —Ç–∏–ø—ñ–∑–æ–≤–∞–Ω–∞** –º–æ–≤–∞. –¶–µ –æ–∑–Ω–∞—á–∞—î, —â–æ –¥–ª—è –∫–æ–∂–Ω–æ—ó –∑–º—ñ–Ω–Ω–æ—ó (—ñ–º–µ–Ω–æ–≤–∞–Ω–æ—ó –∫–æ–º—ñ—Ä–∫–∏ –ø–∞–º'—è—Ç—ñ) –º–∏ –ø–æ–≤–∏–Ω–Ω—ñ –∑–∞–∑–¥–∞–ª–µ–≥—ñ–¥—å –≤–∫–∞–∑–∞—Ç–∏, —è–∫–∏–π —Ç–∏–ø –¥–∞–Ω–∏—Ö –≤–æ–Ω–∞ –±—É–¥–µ –∑–±–µ—Ä—ñ–≥–∞—Ç–∏.

### –§—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω—ñ —Ç–∏–ø–∏ –¥–∞–Ω–∏—Ö

–¶–µ –±–∞–∑–æ–≤—ñ "—Ü–µ–≥–ª–∏–Ω–∫–∏", –∑ —è–∫–∏—Ö –±—É–¥—É—é—Ç—å—Å—è –¥–∞–Ω—ñ —É –ø—Ä–æ–≥—Ä–∞–º–∞—Ö.

### ## –ß–∏—Å–ª–∞ –∑ –ø–ª–∞–≤–∞—é—á–æ—é –∫–æ–º–æ—é: `float` vs. `double`

–û—Å–Ω–æ–≤–Ω–∞ –≤—ñ–¥–º—ñ–Ω–Ω—ñ—Å—Ç—å –º—ñ–∂ `float` (–æ–¥–∏–Ω–∞—Ä–Ω–∞ —Ç–æ—á–Ω—ñ—Å—Ç—å) —Ç–∞ `double` (–ø–æ–¥–≤—ñ–π–Ω–∞ —Ç–æ—á–Ω—ñ—Å—Ç—å) –ø–æ–ª—è–≥–∞—î –≤ **–∫—ñ–ª—å–∫–æ—Å—Ç—ñ –ø–∞–º'—è—Ç—ñ**, —è–∫—É –≤–æ–Ω–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å, —â–æ –±–µ–∑–ø–æ—Å–µ—Ä–µ–¥–Ω—å–æ –≤–ø–ª–∏–≤–∞—î –Ω–∞ —ó—Ö–Ω—é **—Ç–æ—á–Ω—ñ—Å—Ç—å** —Ç–∞ **–¥—ñ–∞–ø–∞–∑–æ–Ω** –∑–Ω–∞—á–µ–Ω—å.

* **–†–æ–∑–º—ñ—Ä:** 32 –±—ñ—Ç–∏ (4 –±–∞–π—Ç–∏).
* **–¢–æ—á–Ω—ñ—Å—Ç—å:** –ú–æ–∂–µ —Ç–æ—á–Ω–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–∏—Ç–∏ –ø—Ä–∏–±–ª–∏–∑–Ω–æ **7 –¥–µ—Å—è—Ç–∫–æ–≤–∏—Ö –∑–Ω–∞–∫—ñ–≤**.
* **–ö–æ–ª–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏:** –£ –≤–∏–ø–∞–¥–∫–∞—Ö, –∫–æ–ª–∏ –≤–∏—Å–æ–∫–∞ —Ç–æ—á–Ω—ñ—Å—Ç—å –Ω–µ —î –∫—Ä–∏—Ç–∏—á–Ω–æ—é, –∞ –µ–∫–æ–Ω–æ–º—ñ—è –ø–∞–º'—è—Ç—ñ –≤–∞–∂–ª–∏–≤–∞. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, —É –∫–æ–º–ø'—é—Ç–µ—Ä–Ω—ñ–π –≥—Ä–∞—Ñ—ñ—Ü—ñ –¥–ª—è –∑–±–µ—Ä—ñ–≥–∞–Ω–Ω—è –∫–æ–æ—Ä–¥–∏–Ω–∞—Ç –∞–±–æ –≤ –¥–µ—è–∫–∏—Ö —Ñ—ñ–∑–∏—á–Ω–∏—Ö —Å–∏–º—É–ª—è—Ü—ñ—è—Ö.

#### `double` (–ü–æ–¥–≤—ñ–π–Ω–∞ —Ç–æ—á–Ω—ñ—Å—Ç—å)
* **–†–æ–∑–º—ñ—Ä:** 64 –±—ñ—Ç–∏ (8 –±–∞–π—Ç).
* **–¢–æ—á–Ω—ñ—Å—Ç—å:** –ú–æ–∂–µ —Ç–æ—á–Ω–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–∏—Ç–∏ –ø—Ä–∏–±–ª–∏–∑–Ω–æ **15-17 –¥–µ—Å—è—Ç–∫–æ–≤–∏—Ö –∑–Ω–∞–∫—ñ–≤**.
* **–ö–æ–ª–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏:** –¶–µ —Ç–∏–ø –∑–∞ –∑–∞–º–æ–≤—á—É–≤–∞–Ω–Ω—è–º –¥–ª—è —Ä–æ–±–æ—Ç–∏ –∑ –¥—ñ–π—Å–Ω–∏–º–∏ —á–∏—Å–ª–∞–º–∏ –≤ C++. –ô–æ–≥–æ —Å–ª—ñ–¥ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ —É –±—ñ–ª—å—à–æ—Å—Ç—ñ –≤–∏–ø–∞–¥–∫—ñ–≤, –æ—Å–æ–±–ª–∏–≤–æ –≤ –Ω–∞—É–∫–æ–≤–∏—Ö, —Ñ—ñ–Ω–∞–Ω—Å–æ–≤–∏—Ö —Ç–∞ —ñ–Ω–∂–µ–Ω–µ—Ä–Ω–∏—Ö —Ä–æ–∑—Ä–∞—Ö—É–Ω–∫–∞—Ö, –¥–µ —Ç–æ—á–Ω—ñ—Å—Ç—å —î –ø—Ä—ñ–æ—Ä–∏—Ç–µ—Ç–æ–º.

> **–ê–Ω–∞–ª–æ–≥—ñ—è:** üí° –£—è–≤—ñ—Ç—å, —â–æ `float` ‚Äî —Ü–µ –∑–≤–∏—á–∞–π–Ω–∞ –ª—ñ–Ω—ñ–π–∫–∞, —è–∫–∞ –¥–æ–∑–≤–æ–ª—è—î –≤–∏–º—ñ—Ä—è—Ç–∏ –¥–æ–≤–∂–∏–Ω—É –∑ —Ç–æ—á–Ω—ñ—Å—Ç—é –¥–æ –º—ñ–ª—ñ–º–µ—Ç—Ä–∞. –î–ª—è –±—ñ–ª—å—à–æ—Å—Ç—ñ –ø–æ–≤—Å—è–∫–¥–µ–Ω–Ω–∏—Ö –∑–∞–≤–¥–∞–Ω—å —Ü—å–æ–≥–æ –¥–æ—Å—Ç–∞—Ç–Ω—å–æ. `double` ‚Äî —Ü–µ –ª–∞–∑–µ—Ä–Ω–∏–π –≤–∏–º—ñ—Ä—é–≤–∞—á, —â–æ –¥–∞—î –∑–Ω–∞—á–Ω–æ —Ç–æ—á–Ω—ñ—à–∏–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç, —è–∫–∏–π —î –∫—Ä–∏—Ç–∏—á–Ω–∏–º –¥–ª—è –ø—Ä–æ—Ñ–µ—Å—ñ–π–Ω–∏—Ö —Ä–æ–∑—Ä–∞—Ö—É–Ω–∫—ñ–≤.

## 4\. –í–≤—ñ–¥ —Ç–∞ –≤–∏–≤—ñ–¥ –¥–∞–Ω–∏—Ö (I/O)

### –í–∏–≤—ñ–¥ –¥–∞–Ω–∏—Ö –∑ `cout`

### ## `endl` —á–∏ `'\n'`: –£ —á–æ–º—É –Ω–∞—Å–ø—Ä–∞–≤–¥—ñ —Ä—ñ–∑–Ω–∏—Ü—è?

–©–æ–± –∑—Ä–æ–∑—É–º—ñ—Ç–∏ —Ä—ñ–∑–Ω–∏—Ü—é, —Å–ø–µ—Ä—à—É –≤–∞—Ä—Ç–æ –∑–Ω–∞—Ç–∏, —è–∫ —Ä—ñ–∑–Ω—ñ –æ–ø–µ—Ä–∞—Ü—ñ–π–Ω—ñ —Å–∏—Å—Ç–µ–º–∏ –ø–æ–∑–Ω–∞—á–∞—é—Ç—å –∫—ñ–Ω–µ—Ü—å —Ä—è–¥–∫–∞:

–°–∏–º–≤–æ–ª `'\n'` —É –≤–∞—à–æ–º—É C++ –∫–æ–¥—ñ ‚Äî —Ü–µ —É–Ω—ñ–≤–µ—Ä—Å–∞–ª—å–Ω–∏–π –∑–∞–ø–∏—Ç –Ω–∞ "–Ω–æ–≤–∏–π —Ä—è–¥–æ–∫". –°—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–∞ –±—ñ–±–ª—ñ–æ—Ç–µ–∫–∞ C++ –∞–≤—Ç–æ–º–∞—Ç–∏—á–Ω–æ –ø–µ—Ä–µ—Ç–≤–æ—Ä—é—î —Ü–µ–π `'\n'` –Ω–∞ –ø—Ä–∞–≤–∏–ª—å–Ω—É –ø–æ—Å–ª—ñ–¥–æ–≤–Ω—ñ—Å—Ç—å –¥–ª—è —Ç—ñ—î—ó –æ–ø–µ—Ä–∞—Ü—ñ–π–Ω–æ—ó —Å–∏—Å—Ç–µ–º–∏, –ø—ñ–¥ —è–∫—É –≤–∏ –∫–æ–º–ø—ñ–ª—é—î—Ç–µ –ø—Ä–æ–≥—Ä–∞–º—É. –¢–æ–±—Ç–æ, `cout << '\n';` –Ω–∞ Windows –≤–∏–≤–µ–¥–µ `\r\n`, –∞ –Ω–∞ Linux ‚Äî `\n`.

–¢–µ–ø–µ—Ä —Ä–æ–∑–≥–ª—è–Ω–µ–º–æ `endl`. **`std::endl`** ‚Äî —Ü–µ –º–∞–Ω—ñ–ø—É–ª—è—Ç–æ—Ä, —è–∫–∏–π –≤–∏–∫–æ–Ω—É—î **–¥–≤—ñ –¥—ñ—ó**:
1.  –í—Å—Ç–∞–≤–ª—è—î —Å–∏–º–≤–æ–ª –Ω–æ–≤–æ–≥–æ —Ä—è–¥–∫–∞ `'\n'` (—è–∫–∏–π, —è–∫ –º–∏ –∑'—è—Å—É–≤–∞–ª–∏, –∞–≤—Ç–æ–º–∞—Ç–∏—á–Ω–æ –∞–¥–∞–ø—Ç—É—î—Ç—å—Å—è –¥–æ –û–°).

### –í–≤—ñ–¥ –¥–∞–Ω–∏—Ö –∑ `cin`

–î–ª—è –æ—Ç—Ä–∏–º–∞–Ω–Ω—è –¥–∞–Ω–∏—Ö –≤—ñ–¥ –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞ (–∑–∞–∑–≤–∏—á–∞–π –∑ –∫–ª–∞–≤—ñ–∞—Ç—É—Ä–∏) –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–∏–π –ø–æ—Ç—ñ–∫ –≤–≤–æ–¥—É `cin` —Ç–∞ –æ–ø–µ—Ä–∞—Ç–æ—Ä –≤–∏–ª—É—á–µ–Ω–Ω—è –∑ –ø–æ—Ç–æ–∫—É `>>`.
