    // –ü—Ä–æ—Å—Ç—ñ—à–µ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞—Ç–∏ find –¥–ª—è –ø–æ—á–∞—Ç–∫—É –∑–Ω–∞—á–µ–Ω–Ω—è:
