## –©–æ —Ç–∞–∫–µ –ø–æ—Å–∏–ª–∞–Ω–Ω—è (Reference)?

**–ü–æ—Å–∏–ª–∞–Ω–Ω—è (Reference)**, —è–∫–µ –∑'—è–≤–∏–ª–æ—Å—è –≤ C++, ‚Äî —Ü–µ **–ø—Å–µ–≤–¥–æ–Ω—ñ–º (alias)** –∞–±–æ –∞–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤–Ω–µ —ñ–º'—è –¥–ª—è *–≤–∂–µ —ñ—Å–Ω—É—é—á–æ—ó* –∑–º—ñ–Ω–Ω–æ—ó. –ü—ñ—Å–ª—è —Å—Ç–≤–æ—Ä–µ–Ω–Ω—è –ø–æ—Å–∏–ª–∞–Ω–Ω—è, –≤–∏ –º–æ–∂–µ—Ç–µ –∑–≤–µ—Ä—Ç–∞—Ç–∏—Å—è –¥–æ –æ–¥–Ω—ñ—î—ó —ñ —Ç—ñ—î—ó –∂ –∫–æ–º—ñ—Ä–∫–∏ –ø–∞–º'—è—Ç—ñ, –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—á–∏ —è–∫ –æ—Ä–∏–≥—ñ–Ω–∞–ª—å–Ω–µ —ñ–º'—è, —Ç–∞–∫ —ñ —ñ–º'—è-–ø—Å–µ–≤–¥–æ–Ω—ñ–º.

–û—Å–Ω–æ–≤–Ω–µ –ø—Ä–∏–∑–Ω–∞—á–µ–Ω–Ω—è –ø–æ—Å–∏–ª–∞–Ω—å ‚Äî –≤–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è –≤ —è–∫–æ—Å—Ç—ñ –ø–∞—Ä–∞–º–µ—Ç—Ä—ñ–≤ —Ñ—É–Ω–∫—Ü—ñ–π –¥–ª—è —Ä–µ–∞–ª—ñ–∑–∞—Ü—ñ—ó –º–µ—Ö–∞–Ω—ñ–∑–º—É **–ø–µ—Ä–µ–¥–∞—á—ñ –∑–∞ –ø–æ—Å–∏–ª–∞–Ω–Ω—è–º (pass-by-reference)**.

### –ü–æ–¥–≤—ñ–π–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è —Å–∏–º–≤–æ–ª—É `&`

–°–∏–º–≤–æ–ª `&` –≤ C++ –º–∞—î –¥–≤–∞ –∑–Ω–∞—á–µ–Ω–Ω—è –∑–∞–ª–µ–∂–Ω–æ –≤—ñ–¥ –∫–æ–Ω—Ç–µ–∫—Å—Ç—É:

  * **–í –≤–∏—Ä–∞–∑–∞—Ö (RHS):** –¶–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä **–æ—Ç—Ä–∏–º–∞–Ω–Ω—è –∞–¥—Ä–µ—Å–∏**. `&number` –ø–æ–≤–µ—Ä—Ç–∞—î –∞–¥—Ä–µ—Å—É –∑–º—ñ–Ω–Ω–æ—ó `number`.
  * **–í –æ–≥–æ–ª–æ—à–µ–Ω–Ω—è—Ö (LHS):** –¶–µ —á–∞—Å—Ç–∏–Ω–∞ —ñ–¥–µ–Ω—Ç–∏—Ñ—ñ–∫–∞—Ç–æ—Ä–∞ —Ç–∏–ø—É, —â–æ —Å—Ç–≤–æ—Ä—é—î **–ø–æ—Å–∏–ª–∞–Ω–Ω—è**. `int &refNumber = number;` —Å—Ç–≤–æ—Ä—é—î –ø–æ—Å–∏–ª–∞–Ω–Ω—è `refNumber` –Ω–∞ –∑–º—ñ–Ω–Ω—É `number`.

**–°–∏–Ω—Ç–∞–∫—Å–∏—Å –æ–≥–æ–ª–æ—à–µ–Ω–Ω—è:**

```cpp
// newName —Å—Ç–∞—î –Ω–æ–≤–∏–º —ñ–º'—è–º (–ø—Å–µ–≤–¥–æ–Ω—ñ–º–æ–º) –¥–ª—è existingName

// –ü—Ä–∏–∫–ª–∞–¥:

int &refNumber = number; // refNumber - —Ü–µ —Ç–µ–ø–µ—Ä —ñ–Ω—à–µ —ñ–º'—è –¥–ª—è 'number'

refNumber = 20; // –¶–µ –∑–º—ñ–Ω—é—î 'number' –Ω–∞ 20

```

–í–Ω—É—Ç—Ä—ñ—à–Ω—å–æ –ø–æ—Å–∏–ª–∞–Ω–Ω—è –ø—Ä–∞—Ü—é—î –ø–æ–¥—ñ–±–Ω–æ –¥–æ –≤–∫–∞–∑—ñ–≤–Ω–∏–∫–∞: –≤–æ–Ω–æ –∑–±–µ—Ä—ñ–≥–∞—î –∞–¥—Ä–µ—Å—É –∑–º—ñ–Ω–Ω–æ—ó, –Ω–∞ —è–∫—É –ø–æ—Å–∏–ª–∞—î—Ç—å—Å—è.

## –ü–µ—Ä–µ–¥–∞—á–∞ –ø–∞—Ä–∞–º–µ—Ç—Ä—ñ–≤ —É —Ñ—É–Ω–∫—Ü—ñ—ó

–¶–µ –∫–ª—é—á–æ–≤–µ –∑–∞—Å—Ç–æ—Å—É–≤–∞–Ω–Ω—è –ø–æ—Å–∏–ª–∞–Ω—å. –Ü—Å–Ω—É—î —Ç—Ä–∏ –æ—Å–Ω–æ–≤–Ω—ñ —Å–ø–æ—Å–æ–±–∏ –ø–µ—Ä–µ–¥–∞—á—ñ –ø–∞—Ä–∞–º–µ—Ç—Ä—ñ–≤ —É —Ñ—É–Ω–∫—Ü—ñ—é:

#### 1\. –ó–∞ –∑–Ω–∞—á–µ–Ω–Ω—è–º (Pass-by-Value)

  * **–©–æ –≤—ñ–¥–±—É–≤–∞—î—Ç—å—Å—è:** –£ —Ñ—É–Ω–∫—Ü—ñ—é –ø–µ—Ä–µ–¥–∞—î—Ç—å—Å—è **–∫–æ–ø—ñ—è** –∞—Ä–≥—É–º–µ–Ω—Ç—É.
  * **–ï—Ñ–µ–∫—Ç:** –£—Å—ñ –∑–º—ñ–Ω–∏ –≤—Å–µ—Ä–µ–¥–∏–Ω—ñ —Ñ—É–Ω–∫—Ü—ñ—ó –≤—ñ–¥–±—É–≤–∞—é—Ç—å—Å—è –∑ –ª–æ–∫–∞–ª—å–Ω–æ—é –∫–æ–ø—ñ—î—é —ñ **–Ω–µ –≤–ø–ª–∏–≤–∞—é—Ç—å** –Ω–∞ –æ—Ä–∏–≥—ñ–Ω–∞–ª—å–Ω—É –∑–º—ñ–Ω–Ω—É.
  * **–ü—Ä–∏–∫–ª–∞–¥:** `void func(int x);`
  * **–ù–µ–¥–æ–ª—ñ–∫:** –î—É–∂–µ –Ω–µ–µ—Ñ–µ–∫—Ç–∏–≤–Ω–æ, —è–∫—â–æ `x` ‚Äî —Ü–µ –≤–µ–ª–∏–∫–∏–π –æ–±'—î–∫—Ç (–Ω–∞–ø—Ä–∏–∫–ª–∞–¥, `string` –∞–±–æ –≤–µ–∫—Ç–æ—Ä), –æ—Å–∫—ñ–ª—å–∫–∏ —Å—Ç–≤–æ—Ä–µ–Ω–Ω—è –∫–æ–ø—ñ—ó –∑–∞–π–º–∞—î —á–∞—Å —ñ –ø–∞–º'—è—Ç—å.

#### 2\. –ó–∞ –≤–∫–∞–∑—ñ–≤–Ω–∏–∫–æ–º (Pass-by-Pointer) - *–°—Ç–∏–ª—å C*

  * **–©–æ –≤—ñ–¥–±—É–≤–∞—î—Ç—å—Å—è:** –£ —Ñ—É–Ω–∫—Ü—ñ—é –ø–µ—Ä–µ–¥–∞—î—Ç—å—Å—è **–∞–¥—Ä–µ—Å–∞** –∑–º—ñ–Ω–Ω–æ—ó (–∫–æ–ø—ñ—è –≤–∫–∞–∑—ñ–≤–Ω–∏–∫–∞).
  * **–ï—Ñ–µ–∫—Ç:** –§—É–Ω–∫—Ü—ñ—è –º–æ–∂–µ –∑–º—ñ–Ω—é–≤–∞—Ç–∏ –æ—Ä–∏–≥—ñ–Ω–∞–ª—å–Ω—É –∑–º—ñ–Ω–Ω—É, —Ä–æ–∑—ñ–º–µ–Ω–æ–≤—É—é—á–∏ –≤–∫–∞–∑—ñ–≤–Ω–∏–∫.
  * **–°–∏–Ω—Ç–∞–∫—Å–∏—Å:** –í–∏–º–∞–≥–∞—î `*` —Ç–∞ `&` —É –∫–æ–¥—ñ, —â–æ —Ä–æ–±–∏—Ç—å –π–æ–≥–æ –º–µ–Ω—à —á–∏—Ç–∞–±–µ–ª—å–Ω–∏–º.
  * **–ü—Ä–∏–∫–ª–∞–¥:** `void func(int* x) { *x = 10; }` // –í–∏–∫–ª–∏–∫: `func(&myVar);`

#### 3\. –ó–∞ –ø–æ—Å–∏–ª–∞–Ω–Ω—è–º (Pass-by-Reference) - *–°—Ç–∏–ª—å C++*
