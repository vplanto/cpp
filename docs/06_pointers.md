## LHS —Ç–∞ RHS: –õ—ñ–≤–∞ —Ç–∞ –ü—Ä–∞–≤–∞ —Å—Ç–æ—Ä–æ–Ω–∏ –æ–ø–µ—Ä–∞—Ü—ñ—ó

–£ –º–æ–≤–∞—Ö –ø—Ä–æ–≥—Ä–∞–º—É–≤–∞–Ω–Ω—è, –æ—Å–æ–±–ª–∏–≤–æ –ø—Ä–∏ —Ä–æ–±–æ—Ç—ñ –∑ –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–º –ø—Ä–∏—Å–≤–æ—î–Ω–Ω—è (`=`), –ø–æ–Ω—è—Ç—Ç—è **LHS (Left-Hand Side)** —Ç–∞ **RHS (Right-Hand Side)** —î —Ñ—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω–∏–º–∏. –í–æ–Ω–∏ –æ–∑–Ω–∞—á–∞—é—Ç—å –ª—ñ–≤—É —Ç–∞ –ø—Ä–∞–≤—É —Å—Ç–æ—Ä–æ–Ω–∏ –≤—ñ–¥ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ —ñ –º–∞—é—Ç—å –∞–±—Å–æ–ª—é—Ç–Ω–æ —Ä—ñ–∑–Ω—ñ —Ä–æ–ª—ñ —Ç–∞ –ø—Ä–∞–≤–∏–ª–∞.

### RHS (–ü—Ä–∞–≤–∞ —Å—Ç–æ—Ä–æ–Ω–∞) ‚Äî –°—Ç–æ—Ä–æ–Ω–∞ "–ó–Ω–∞—á–µ–Ω–Ω—è"

**–†–æ–ª—å:** –û—Å–Ω–æ–≤–Ω–∞ –∑–∞–¥–∞—á–∞ –ø—Ä–∞–≤–æ—ó —Å—Ç–æ—Ä–æ–Ω–∏ ‚Äî **–±—É—Ç–∏ –æ–±—á–∏—Å–ª–µ–Ω–æ—é (evaluated) –¥–æ —î–¥–∏–Ω–æ–≥–æ, –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ–≥–æ –∑–Ω–∞—á–µ–Ω–Ω—è.** –¶–µ —Å—Ç–æ—Ä–æ–Ω–∞, —è–∫–∞ "–¥–∞—î" —Ä–µ–∑—É–ª—å—Ç–∞—Ç.

**–ß–∏–º –º–æ–∂–µ –±—É—Ç–∏ RHS:**

  * **–õ—ñ—Ç–µ—Ä–∞–ª–æ–º:** `x = 10;` (RHS ‚Äî —Ü–µ —á–∏—Å–ª–æ `10`)
  * **–ó–º—ñ–Ω–Ω–æ—é:** `x = y;` (RHS ‚Äî —Ü–µ –∑–Ω–∞—á–µ–Ω–Ω—è, —â–æ –∑–±–µ—Ä—ñ–≥–∞—î—Ç—å—Å—è —É –∑–º—ñ–Ω–Ω—ñ–π `y`)
  * **–ê—Ä–∏—Ñ–º–µ—Ç–∏—á–Ω–∏–º –≤–∏—Ä–∞–∑–æ–º:** `x = y + z * 2;` (RHS ‚Äî —Ü–µ —Ä–µ–∑—É–ª—å—Ç–∞—Ç –æ–±—á–∏—Å–ª–µ–Ω–Ω—è –≤–∏—Ä–∞–∑—É)
  * **–í–∏–∫–ª–∏–∫–æ–º —Ñ—É–Ω–∫—Ü—ñ—ó, —â–æ –ø–æ–≤–µ—Ä—Ç–∞—î –∑–Ω–∞—á–µ–Ω–Ω—è:** `x = getArea(5.0);` (RHS ‚Äî —Ü–µ –∑–Ω–∞—á–µ–Ω–Ω—è, —è–∫–µ –ø–æ–≤–µ—Ä–Ω–µ —Ñ—É–Ω–∫—Ü—ñ—è `getArea`)
  * **–û–ø–µ—Ä–∞—Ç–æ—Ä–æ–º `&`:** `ptr = &x;` (RHS ‚Äî —Ü–µ –∞–¥—Ä–µ—Å–∞ –∑–º—ñ–Ω–Ω–æ—ó `x`)

> **–ê–Ω–∞–ª–æ–≥—ñ—è:** –£—è–≤—ñ—Ç—å, —â–æ RHS ‚Äî —Ü–µ –º–∞—Ç–µ–º–∞—Ç–∏—á–Ω–∞ –∑–∞–¥–∞—á–∞ –Ω–∞ –¥–æ—à—Ü—ñ. –ü–µ—Ä—à –Ω—ñ–∂ —â–æ—Å—å —Ä–æ–±–∏—Ç–∏ –¥–∞–ª—ñ, –≤–∞–º –ø–æ—Ç—Ä—ñ–±–Ω–æ —ó—ó —Ä–æ–∑–≤'—è–∑–∞—Ç–∏ —ñ –æ—Ç—Ä–∏–º–∞—Ç–∏ –æ–¥–Ω—É —á—ñ—Ç–∫—É –≤—ñ–¥–ø–æ–≤—ñ–¥—å.

### LHS (–õ—ñ–≤–∞ —Å—Ç–æ—Ä–æ–Ω–∞) ‚Äî –°—Ç–æ—Ä–æ–Ω–∞ "–ü—Ä–∏–∑–Ω–∞—á–µ–Ω–Ω—è"

**–†–æ–ª—å:** –û—Å–Ω–æ–≤–Ω–∞ –∑–∞–¥–∞—á–∞ –ª—ñ–≤–æ—ó —Å—Ç–æ—Ä–æ–Ω–∏ ‚Äî **–≤–∏–∑–Ω–∞—á–∏—Ç–∏ –º—ñ—Å—Ü–µ –≤ –ø–∞–º'—è—Ç—ñ (–±—É—Ç–∏ –ª–æ–∫–∞—Ç–æ—Ä–æ–º), –∫—É–¥–∏ –±—É–¥–µ –∑–±–µ—Ä–µ–∂–µ–Ω–æ —Ä–µ–∑—É–ª—å—Ç–∞—Ç**, –æ—Ç—Ä–∏–º–∞–Ω–∏–π –∑ –ø—Ä–∞–≤–æ—ó —Å—Ç–æ—Ä–æ–Ω–∏.

**–ß–∏–º –º–æ–∂–µ –±—É—Ç–∏ LHS (–≤ –æ–ø–µ—Ä–∞—Ü—ñ—ó –ø—Ä–∏—Å–≤–æ—î–Ω–Ω—è):**

  * **–ó–º—ñ–Ω–Ω–æ—é:** `x = 10;` (LHS ‚Äî —Ü–µ –∫–æ–º—ñ—Ä–∫–∞ –ø–∞–º'—è—Ç—ñ, —â–æ –∞—Å–æ—Ü—ñ–π–æ–≤–∞–Ω–∞ –∑—ñ –∑–º—ñ–Ω–Ω–æ—é `x`)
  * **–ï–ª–µ–º–µ–Ω—Ç–æ–º –º–∞—Å–∏–≤—É:** `arr[0] = 20;`
  * **–†–æ–∑—ñ–º–µ–Ω–æ–≤–∞–Ω–∏–º –≤–∫–∞–∑—ñ–≤–Ω–∏–∫–æ–º:** `*ptr = 30;`

**–í–∞–∂–ª–∏–≤–æ:** LHS –Ω–µ –º–æ–∂–µ –±—É—Ç–∏ –≤–∏—Ä–∞–∑–æ–º –∞–±–æ –ª—ñ—Ç–µ—Ä–∞–ª–æ–º. –°–∞–º–µ —Ç–æ–º—É –∑–∞–ø–∏—Å `x + y = 10;` —î **–Ω–µ–∫–æ—Ä–µ–∫—Ç–Ω–∏–º** —É –ø—Ä–æ–≥—Ä–∞–º—É–≤–∞–Ω–Ω—ñ. –í–∏—Ä–∞–∑ `x + y` –æ–±—á–∏—Å–ª—é—î—Ç—å—Å—è —É –∑–Ω–∞—á–µ–Ω–Ω—è, –∞–ª–µ –≤—ñ–Ω –Ω–µ —î –∫–æ–º—ñ—Ä–∫–æ—é –ø–∞–º'—è—Ç—ñ, –∫—É–¥–∏ –º–æ–∂–Ω–∞ —â–æ—Å—å –∑–∞–ø–∏—Å–∞—Ç–∏.

> **–ê–Ω–∞–ª–æ–≥—ñ—è:** –£—è–≤—ñ—Ç—å, —â–æ LHS ‚Äî —Ü–µ –ø–æ—à—Ç–æ–≤–∞ —Å–∫—Ä–∏–Ω—å–∫–∞ –∑ –Ω–æ–º–µ—Ä–æ–º. –í–∏ –Ω–µ –º–æ–∂–µ—Ç–µ –≤—ñ–¥–ø—Ä–∞–≤–∏—Ç–∏ –ª–∏—Å—Ç "–Ω–∞ –≤—ñ–¥–ø–æ–≤—ñ–¥—å –∑–∞–¥–∞—á—ñ 2+2". –í–∏ –º–æ–∂–µ—Ç–µ –≤—ñ–¥–ø—Ä–∞–≤–∏—Ç–∏ –π–æ–≥–æ –ª–∏—à–µ –Ω–∞ –∫–æ–Ω–∫—Ä–µ—Ç–Ω—É –∞–¥—Ä–µ—Å—É, –Ω–∞–ø—Ä–∏–∫–ª–∞–¥, "—Å–∫—Ä–∏–Ω—å–∫–∞ ‚Ññ4".

### –ü—ñ–¥—Å—É–º–∫–æ–≤–∞ —Ç–∞–±–ª–∏—Ü—è

| **–û—Å–Ω–æ–≤–Ω–∞ —Ä–æ–ª—å** | –ü—Ä–∏–∑–Ω–∞—á–µ–Ω–Ω—è (Destination / Locator) | –ó–Ω–∞—á–µ–Ω–Ω—è (Value / Source) |
| **–©–æ —Ä–æ–±–∏—Ç—å?** | –í–∫–∞–∑—É—î, **–∫—É–¥–∏** –∑–±–µ—Ä–µ–≥—Ç–∏ —Ä–µ–∑—É–ª—å—Ç–∞—Ç. | –û–±—á–∏—Å–ª—é—î—Ç—å—Å—è, **—â–æ** —Å–∞–º–µ –∑–±–µ—Ä–µ–≥—Ç–∏. |

### –û–ø–µ—Ä–∞—Ç–æ—Ä —Ä–æ–∑—ñ–º–µ–Ω—É–≤–∞–Ω–Ω—è `*`

–©–æ–± –æ—Ç—Ä–∏–º–∞—Ç–∏ –∞–±–æ –∑–º—ñ–Ω–∏—Ç–∏ **–∑–Ω–∞—á–µ–Ω–Ω—è, —â–æ –∑–±–µ—Ä—ñ–≥–∞—î—Ç—å—Å—è –∑–∞ –∞–¥—Ä–µ—Å–æ—é**, —è–∫—É —Ç—Ä–∏–º–∞—î –≤–∫–∞–∑—ñ–≤–Ω–∏–∫, –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è **–æ–ø–µ—Ä–∞—Ç–æ—Ä —Ä–æ–∑—ñ–º–µ–Ω—É–≤–∞–Ω–Ω—è `*`**.

```cpp
int number = 88;
int *pNumber = &number;

// –í–∏–≤—ñ–¥ —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—ó
cout << pNumber << endl;  // –í–∏–≤–µ–¥–µ –∞–¥—Ä–µ—Å—É –∑–º—ñ–Ω–Ω–æ—ó number (–Ω–∞–ø—Ä., 0x22ccec)
cout << *pNumber << endl; // –í–∏–≤–µ–¥–µ –∑–Ω–∞—á–µ–Ω–Ω—è, —â–æ –ª–µ–∂–∏—Ç—å –∑–∞ —Ü—ñ—î—é –∞–¥—Ä–µ—Å–æ—é (88)

// –ó–º—ñ–Ω–∞ –∑–Ω–∞—á–µ–Ω–Ω—è —á–µ—Ä–µ–∑ –≤–∫–∞–∑—ñ–≤–Ω–∏–∫
*pNumber = 99; // –ó–∞–ø–∏—Å–∞—Ç–∏ –∑–Ω–∞—á–µ–Ω–Ω—è 99 –≤ –∫–æ–º—ñ—Ä–∫—É, –Ω–∞ —è–∫—É –≤–∫–∞–∑—É—î pNumber

cout << *pNumber << endl; // –í–∏–≤–µ–¥–µ 99
cout << number << endl;   // –ó–Ω–∞—á–µ–Ω–Ω—è –æ—Ä–∏–≥—ñ–Ω–∞–ª—å–Ω–æ—ó –∑–º—ñ–Ω–Ω–æ—ó —Ç–∞–∫–æ–∂ –∑–º—ñ–Ω–∏–ª–æ—Å—è –Ω–∞ 99
```

### ‚ùì –ü–∏—Ç–∞–Ω–Ω—è –¥–æ –≥—Ä—É–ø–∏

–£ –Ω–∞—Å —î –¥–≤–∞ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∏, —è–∫—ñ –ø–æ–∑–Ω–∞—á–∞—é—Ç—å—Å—è `*`:

1.  –û–≥–æ–ª–æ—à–µ–Ω–Ω—è: `int *ptr;`
2.  –†–æ–∑—ñ–º–µ–Ω—É–≤–∞–Ω–Ω—è: `*ptr = 99;`
    –ê —Ç–µ–ø–µ—Ä –ø–æ–¥–∏–≤—ñ—Ç—å—Å—è –Ω–∞ `*pNumber = 99;` —Ç–∞ `number = 99;`. –í–æ–Ω–∏ —Ä–æ–±–ª—è—Ç—å —Ç–µ —Å–∞–º–µ. –ù–∞–≤—ñ—â–æ –Ω–∞–º –≤–∑–∞–≥–∞–ª—ñ `*pNumber`, —è–∫—â–æ —î `number`? –î–µ —Ü–µ –º–æ–∂–µ –±—É—Ç–∏ –∫–æ—Ä–∏—Å–Ω–∏–º? (–ü—ñ–¥–∫–∞–∑–∫–∞: –∑–≥–∞–¥–∞–π—Ç–µ –ø—Ä–æ —Ñ—É–Ω–∫—Ü—ñ—ó).

–¶–µ –∫–æ—Ä–∏—Å–Ω–æ, –∫–æ–ª–∏ –º–∏ —Ö–æ—á–µ–º–æ **–¥–∞—Ç–∏ —Ñ—É–Ω–∫—Ü—ñ—ó –º–æ–∂–ª–∏–≤—ñ—Å—Ç—å –∑–º—ñ–Ω–∏—Ç–∏ –Ω–∞—à—É –∑–º—ñ–Ω–Ω—É**. –ú–∏ –Ω–µ –º–æ–∂–µ–º–æ –ø–µ—Ä–µ–¥–∞—Ç–∏ `number` –≤ —ñ–Ω—à—É —Ñ—É–Ω–∫—Ü—ñ—é —ñ –∑–º—ñ–Ω–∏—Ç–∏ —ó—ó (—Ü–µ –±—É–¥–µ pass-by-value, C++ —Å—Ç–≤–æ—Ä–∏—Ç—å –∫–æ–ø—ñ—é). –ê–ª–µ –º–∏ –º–æ–∂–µ–º–æ –ø–µ—Ä–µ–¥–∞—Ç–∏ *–∞–¥—Ä–µ—Å—É* `number` (—Ç–æ–±—Ç–æ `pNumber`). –§—É–Ω–∫—Ü—ñ—è, –æ—Ç—Ä–∏–º–∞–≤—à–∏ –∞–¥—Ä–µ—Å—É, –º–æ–∂–µ "–¥–æ—Ç—è–≥–Ω—É—Ç–∏—Å—è" –¥–æ –Ω–∞—à–æ—ó `number` –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ `*` —ñ –∑–º—ñ–Ω–∏—Ç–∏ —ó—ó. –¶–µ –Ω–∞–∑–∏–≤–∞—î—Ç—å—Å—è **pass-by-pointer**.

## –ü—Ä–∞–≤–∏–ª–∞ —Ç–∞ –Ω–∞–π—á–∞—Å—Ç—ñ—à—ñ –ø–æ–º–∏–ª–∫–∏

### –°—É–≤–æ—Ä–∞ —Ç–∏–ø—ñ–∑–∞—Ü—ñ—è –≤–∫–∞–∑—ñ–≤–Ω–∏–∫—ñ–≤

–í–∫–∞–∑—ñ–≤–Ω–∏–∫ —î —Å—Ç—Ä–æ–≥–æ —Ç–∏–ø—ñ–∑–æ–≤–∞–Ω–∏–º. –í–∫–∞–∑—ñ–≤–Ω–∏–∫ –Ω–∞ `int` –º–æ–∂–µ –∑–±–µ—Ä—ñ–≥–∞—Ç–∏ –ª–∏—à–µ –∞–¥—Ä–µ—Å—É –∑–º—ñ–Ω–Ω–æ—ó `int`, –≤–∫–∞–∑—ñ–≤–Ω–∏–∫ –Ω–∞ `double` ‚Äî –ª–∏—à–µ –∞–¥—Ä–µ—Å—É `double`.

// iPtr = &d; // –ü–û–ú–ò–õ–ö–ê: –Ω–µ –º–æ–∂–Ω–∞ –ø—Ä–∏—Å–≤–æ—ó—Ç–∏ –∞–¥—Ä–µ—Å—É double –≤–∫–∞–∑—ñ–≤–Ω–∏–∫—É –Ω–∞ int

### –ù–µ—ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–æ–≤–∞–Ω—ñ –≤–∫–∞–∑—ñ–≤–Ω–∏–∫–∏ ("–¥–∏–∫—ñ" –≤–∫–∞–∑—ñ–≤–Ω–∏–∫–∏)

–û–≥–æ–ª–æ—à–µ–Ω–Ω—è –≤–∫–∞–∑—ñ–≤–Ω–∏–∫–∞ –±–µ–∑ —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–∞—Ü—ñ—ó —î –≤–∫—Ä–∞–π –Ω–µ–±–µ–∑–ø–µ—á–Ω–∏–º.

```cpp
int *iPtr;   // iPtr –≤–∫–∞–∑—É—î –Ω–∞ –≤–∏–ø–∞–¥–∫–æ–≤—É –∞–¥—Ä–µ—Å—É –≤ –ø–∞–º'—è—Ç—ñ (–Ω–∞ "—Å–º—ñ—Ç—Ç—è")
*iPtr = 55;  // –ó–∞–ø–∏—Å —á–∏—Å–ª–∞ 55 —É –≤–∏–ø–∞–¥–∫–æ–≤—É –∫–æ–º—ñ—Ä–∫—É!
             // –¶–µ –º–æ–∂–µ "–∑–ª–∞–º–∞—Ç–∏" —ñ–Ω—à—ñ —á–∞—Å—Ç–∏–Ω–∏ –≤–∞—à–æ—ó –ø—Ä–æ–≥—Ä–∞–º–∏ –∞–±–æ –≤—Å—é —Å–∏—Å—Ç–µ–º—É.
```

–¶–µ –æ–¥–Ω–∞ –∑ –Ω–∞–π—Å–µ—Ä–π–æ–∑–Ω—ñ—à–∏—Ö –ø–æ–º–∏–ª–æ–∫, —è–∫—É –∫–æ–º–ø—ñ–ª—è—Ç–æ—Ä –º–æ–∂–µ –Ω–µ –≤–∏—è–≤–∏—Ç–∏.

### –ù—É–ª—å–æ–≤—ñ –≤–∫–∞–∑—ñ–≤–Ω–∏–∫–∏ (Null Pointers)

–©–æ–± —É–Ω–∏–∫–Ω—É—Ç–∏ –ø—Ä–æ–±–ª–µ–º–∏ –∑ "–¥–∏–∫–∏–º–∏" –≤–∫–∞–∑—ñ–≤–Ω–∏–∫–∞–º–∏, —è–∫—â–æ –≤–∫–∞–∑—ñ–≤–Ω–∏–∫ —â–µ –Ω–µ –≤–∫–∞–∑—É—î –Ω–∞ —Ä–µ–∞–ª—å–Ω–∏–π –æ–±'—î–∫—Ç, –π–æ–≥–æ –≤–∞—Ä—Ç–æ —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑—É–≤–∞—Ç–∏ –Ω—É–ª—å–æ–≤–∏–º –∑–Ω–∞—á–µ–Ω–Ω—è–º.

```cpp
int *iPtr = 0;    // –°—Ç–∞—Ä–∏–π —Å—Ç–∏–ª—å C
int *p = NULL;    // –°—Ç–∏–ª—å C, –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î –º–∞–∫—Ä–æ—Å
int *pNew = nullptr; // –°—É—á–∞—Å–Ω–∏–π, —Ä–µ–∫–æ–º–µ–Ω–¥–æ–≤–∞–Ω–∏–π —Å—Ç–∏–ª—å C++11
```

**`nullptr` —î –∫—Ä–∞—â–∏–º –≤–∏–±–æ—Ä–æ–º**, –æ—Å–∫—ñ–ª—å–∫–∏ –≤—ñ–Ω —î —Ç–∏–ø—ñ–∑–æ–≤–∞–Ω–∏–º —ñ –∑–∞–ø–æ–±—ñ–≥–∞—î –ø–µ–≤–Ω–∏–º –≤–∏–¥–∞–º –ø–æ–º–∏–ª–æ–∫.

–°–ø—Ä–æ–±–∞ —Ä–æ–∑—ñ–º–µ–Ω—É–≤–∞—Ç–∏ –Ω—É–ª—å–æ–≤–∏–π –≤–∫–∞–∑—ñ–≤–Ω–∏–∫ (`*pNew`) –ø—Ä–∏–∑–≤–µ–¥–µ –¥–æ –Ω–µ–≥–∞–π–Ω–æ–≥–æ –∞–≤–∞—Ä—ñ–π–Ω–æ–≥–æ –∑–∞–≤–µ—Ä—à–µ–Ω–Ω—è –ø—Ä–æ–≥—Ä–∞–º–∏, —â–æ –Ω–∞–±–∞–≥–∞—Ç–æ –∫—Ä–∞—â–µ, –Ω—ñ–∂ —Ç–∏—Ö–µ –ø–æ—à–∫–æ–¥–∂–µ–Ω–Ω—è –ø–∞–º'—è—Ç—ñ.

## –ö–µ—Ä—É–≤–∞–Ω–Ω—è –¥–æ—Å—Ç—É–ø–æ–º: `const` —Ç–∞ –≤–∫–∞–∑—ñ–≤–Ω–∏–∫–∏
